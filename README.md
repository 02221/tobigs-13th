# Tobigs

## 소개

안녕하세요, 대학생 빅데이터 연합동아리 Tobigs 입니다.

저희는 다양한 전공의 사람들이 빅데이터를 공부하기 위해 모인 동아리이며,

2014년을 시작으로 현재 12기, 13기가 활동 중입니다.

과제 기반 다양한 튜토리얼을 gitbook에 업로드 중입니다.
